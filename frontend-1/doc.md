## 开发环境

- 安装依赖
```
yarn install
```

- 开发模式运行
```
yarn run serve // 启动前端服务
yarn run start // 启动后端服务
```

- 编译项目
```
yarn run build
```


## 作答
1. 作业要求以网页的方式呈现，所以需要开发和部署。
2. 需要解决的问题：  
- csv文件内容读取  
- csv文件内容写入  
- 没有后端接口怎么办  

  解决办法：用node搭建一个后端服务，所有问题基本都不是问题了。  

**csv文件实现数据存储的功能，其他的就是一些增删改查的功能，和数据的处理。没遇到特别难解决不了的，所以这里就不多写了。**
